# ☕ Maison Cafe

A high-end café and kitchen website built with HTML, CSS, and JavaScript. Designed with a luxury aesthetic, snap scroll layout, and a full food & drinks menu.

## 🔗 Live Demo
👉 [View Live Site](https://codecraftstudio-prog.github.io/Maison-cafe/)

---

## 📁 Project Structure

```
Maison-cafe/
├── index.html          # Main HTML file
├── style.css           # All styles
└── images/
    ├── img-table.jpg   # Hero background
    ├── bowl.jpg        # Bento food image
    ├── food.jpg        # Featured menu card
    ├── interior.jpg    # About section
    ├── interior-outdoor.jpg  # Experience section
    └── breakfast.jpg   # Contact section
```

---

## ✨ Features

- **Snap scroll layout** — each section snaps into place like a presentation
- **Split hero** — background image left, food photo + quote right
- **Bento menu grid** — editorial-style with featured dish card
- **Full menu** — Salads, Mains, Bakes & Snacks, Drinks with ₹ pricing
- **About section** — story, stats (6+ origins, 40+ items, 5★)
- **Experience section** — full-screen image with overlay text
- **Contact form** — reservation with occasion dropdown
- **Hours bar** — Open Every Day · 7:00 AM — 10:00 PM
- **Fully responsive** — mobile, tablet, desktop
- **Smooth navbar** — transparent → frosted glass on scroll

## 🚀 How to Run Locally

1. Download the full project folder
2. Make sure `index.html`, `style.css`, and `images/` folder are in the same directory
3. Double-click `index.html` to open in any browser
4. No dependencies or installation needed

---

## ⚠️ Important Note for GitHub Pages

Make sure your repo has this exact structure:
```
Maison-cafe/
├── index.html
├── style.css
└── images/
    └── (all image files here)
```
If images are missing or in a different folder, they won't load on the live site.
